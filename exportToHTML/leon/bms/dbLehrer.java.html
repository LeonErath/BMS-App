49        * @param suchWort      ist das Wort womit der Lehrer in der entsprechenden Kategorie gepseichert ist 
50        * @return gibt den gesucht Lehrer zurück 
51        * @getLehrer abstrakte Methode um einen Lehrer nach bestimmen Kategorien rauszusuchen 
52        */ 
53       public dbLehrer getLehrer(String suchKategorie, String suchWort) { 
54           List<dbLehrer> lehrer = dbLehrer.find(dbLehrer.class, suchKategorie + " = ?", suchWort); 
55           dbLehrer lehrer1 = lehrer.get(0); 
56           return lehrer1; 
57       }
